Hot path Phase 2

Nurse-facing routes return in under 200ms. AI Gateway adds caching + failover so Tier 3 cloud calls stay responsive even if one provider hiccups.

Async lane Phase 3 · 5 · 6

Workflow orchestrates the observation lifecycle. Queues fan out to Sandbox for PDF rendering and heavier ONNX second opinions. Never in the nurse's response time.

Nightly analytics Phase 4

A separate Worker exports Turso to partitioned Parquet on R2. DuckDB powers research and the calibration loop that feeds better models back to the device.
